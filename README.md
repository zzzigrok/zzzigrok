### 🧠 Обо мне

Я **Григорий Романов** — Python & Go разработчик из Москвы. Строю локальные LLM, AI-экосистемы и инструменты, которые «думают» на потребительском железе. Верю в чистый код, открытые архитектуры и силу русского языка в ML.

- 🔭 Сейчас развиваю **Tolstoy AI Studio** — экосистему для обучения и инференса русскоязычных LLM.
- 🌱 В фокусе: **локальные LLM, эффективный инференс, Transformer-архитектуры, GAN**.
- 💬 Спроси меня о: `Python` · `Go` · `LLM training/inference` · `чистой архитектуре`.
