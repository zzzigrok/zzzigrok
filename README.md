### 👨‍💻 Обо мне

Привет! Я **Григорий Романов** — разработчик из Москвы и студент Технологического университета им. Леонова. Я специализируюсь на создании локальных LLM, AI-экосистем и инструментов, которые способны «думать» на потребительском железе. Верю в чистую архитектуру, открытый исходный код и развитие русского языка в машинном обучении.

- 🔭 **Сейчас развиваю:** `Tolstoy AI Studio` — комплексную экосистему для обучения и инференса русскоязычных LLM.
- 🌱 **В фокусе:** локальные LLM (llama.cpp), эффективный инференс (MoE, GQA, RoPE), Transformer-архитектуры и GAN.
- ⚡ **Философия:** *«Building things that think»* — строю то, что умеет рассуждать.
- 🎓 **Академические интересы:** высшая математика, проектирование AI-систем и оптимизация алгоритмов.
- 🛠️ **Инструменты:** активно использую WSL 2, CMake и автоматизацию рабочих процессов через различные API (OpenRouter, Polza.ai).
